# Portfolio Wireframe Template

This is a reusable case-study wireframe extracted from the current portfolio structure.

It keeps:

- Section order and case-study narrative flow
- Layout systems: hero, overview metadata, research methods, users, ideation, testing, product assets, experience flow, five-step walkthrough
- Reusable hierarchy: eyebrow, large heading, lead text, image placeholders, captions, cards, meta grid, step navigation
- Full-width / edge-to-edge section language

It does not keep:

- Project-specific content, branding, images, videos, 3D assets, GLB files, or research data
- Trash Talk, Rumi, Slay, MetroCity, recycling-specific labels, or animation-specific code
- The current hero 3D scene, cinematic object drop sequence, or DotGrid implementation

How to use:

1. Copy the whole `portfolio-wireframe-template` folder into a new project.
2. Replace bracketed placeholder text like `[Project Title]`.
3. Replace `.placeholder` blocks with images, videos, charts, canvas, or embeds.
4. Remove sections you do not need.
5. Keep `object-fit: contain` when adding images that should not be cropped.

Open `index.html` directly in a browser, or serve the folder with any static server.
